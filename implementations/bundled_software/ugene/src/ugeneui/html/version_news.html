Changes in UGENE 1.30

The new version is majorly a bug-fix release.
However, several usability improvements were also added, for example, modification of the selection in the Sequence View. See details on theUGENE web site.
See also anew video about UGENE versions 1.28-1.30.

Изменения в UGENE 1.30

Данная версия преимущественно содержит исправления ошибок.
Также были добавлены несколько улучшений, в частности, изменился подход к выделению аннотаций в редакторе последовательностей. Подробности см. насайте UGENE.
Видеоролик про новые возможности, появившиеся в версиях программы 1.28-1.30, можно посмотреть наканале YouTube (на английском языке).
